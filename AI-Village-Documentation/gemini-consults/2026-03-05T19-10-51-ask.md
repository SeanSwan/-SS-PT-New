To answer your question directly: **We are going to use a Hybrid Approach.** 

Relying solely on CSS keyframes for scroll-linked parallax is a nightmare for maintainability, and relying solely on Framer Motion for *everything* on a mobile device will throttle the main thread, drain battery, and ruin the premium NASM-certified feel we are going for. Think Apple Fitness+ or Peloton—their mobile web experiences feel weightless because they don't fight the browser's native scroll.

Here is my authoritative design and animation blueprint for the Galaxy-Swan Hero Parallax.

---

### 1. The Strategy: Hybrid Motion Architecture

*   **Mobile (320px to 430px):** **Zero scroll-linked parallax.** Mobile users expect 120hz buttery-smooth scrolling. Scroll-jacking or heavy JS listeners here feel cheap. Instead, we use **CSS Keyframes** for an *ambient, continuous "breathing" effect* (like a swan resting on water or a nebula shifting).
*   **Tablet & Desktop (768px to 3840px):** **Framer Motion.** We will utilize Framer Motion’s `useScroll` and `useTransform` to map the scroll position to the `y` axis of our layers. 

*Crucial UX Rule:* We must respect `useReducedMotion()`. If a user has motion sensitivity, all parallax and ambient floating drops to zero, defaulting to a beautiful, static glassmorphic composition.

---

### 2. The Galaxy-Swan Aesthetic Execution

I do not want generic "AI slop" linear gradients or cheap particle effects. We are building a premium, deep-space environment with high-end glassmorphic interfaces.

### 3. Exact Values & Implementation Specs

Claude, when you build the styled-components and Framer Motion logic, use these exact parameters.

#### A. Animation Curves & Timing
*   **Framer Motion Spring (for entrances):** `type: "spring", stiffness: 100, damping: 20, mass: 0.5`
*   **Framer Motion Ease (for parallax mapping):** `ease: [0.16, 1, 0.3, 1]` (This is the Apple fluid ease-out curve).
*   **CSS Ambient Float (Mobile):** `ease-in-out`, `duration: 8s`, `infinite`, `alternate`.

#### B. Parallax Scroll Mapping (Framer Motion)
Map the `scrollY` from `0` to `800px` (typical hero height):
*   **Background Nebula:** `y: [0, 150]` (Moves down slowly, creating depth).
*   **Midground Swan Motifs:** `y: [0, -50]` (Moves up slightly).
*   **Foreground Content:** `y: [0, -150]` (Moves up faster, creating the primary parallax illusion).

```tsx
import React, { useRef } from 'react';
import { motion, useScroll, useTransform, useReducedMotion } from 'framer-motion';
import styled, { keyframes } from 'styled-components';

// --- DESIGN TOKENS ---
const TOKENS = {
  colors: {
    galaxyCore: '#0a0a1a',
    swanCyan: '#00FFFF',
    cosmicPurple: '#7851A9',
  },
  glass: `
    background: rgba(10, 10, 26, 0.4);
    backdrop-filter: blur(24px);
    -webkit-backdrop-filter: blur(24px);
    border: 1px solid rgba(0, 255, 255, 0.15);
    box-shadow: 0 8px 32px 0 rgba(0, 0, 0, 0.37);
  `,
  touchTarget: '44px',
};

// --- CSS KEYFRAMES (Mobile Ambient Motion) ---
const ambientFloat = keyframes`
  0% { transform: translate3d(0, 0px, 0) scale(1); opacity: 0.6; }
  50% { transform: translate3d(0, -15px, 0) scale(1.02); opacity: 0.8; }
  100% { transform: translate3d(0, 0px, 0) scale(1); opacity: 0.6; }
`;

// --- STYLED COMPONENTS ---
const HeroContainer = styled.section`
  position: relative;
  height: 100vh;
  min-height: 800px;
  width: 100%;
  overflow: hidden;
  background-color: ${TOKENS.colors.galaxyCore};
  display: flex;
  align-items: center;
  justify-content: center;
`;

const NebulaBackground = styled(motion.div)`
  position: absolute;
  inset: -20%; /* Bleed edge to prevent clipping during parallax */
  background: radial-gradient(circle at 20% 30%, rgba(120, 81, 169, 0.4) 0%, transparent 40%),
              radial-gradient(circle at 80% 70%, rgba(0, 255, 255, 0.15) 0%, transparent 40%);
  will-change: transform;
  z-index: 0;

  @media (max-width: 767px) {
    animation: ${ambientFloat} 12s ease-in-out infinite alternate;
  }
`;

const ForegroundContent = styled(motion.div)`
  position: relative;
  z-index: 10;
  display: flex;
  flex-direction: column;
  align-items: center;
  text-align: center;
  padding: 0 24px;
  will-change: transform, opacity;
`;

const PrimaryCTA = styled.button`
  ${TOKENS.glass}
  color: #ffffff;
  font-size: 16px;
  font-weight: 600;
  letter-spacing: 0.5px;
  min-height: ${TOKENS.touchTarget}; /* Strict 44px minimum */
  min-width: 200px;
  border-radius: 30px;
  cursor: pointer;
  transition: all 0.3s cubic-bezier(0.16, 1, 0.3, 1);

  &:hover {
    background: rgba(0, 255, 255, 0.1);
    border-color: rgba(0, 255, 255, 0.4);
    box-shadow: 0 0 20px rgba(0, 255, 255, 0.2);
    transform: translateY(-2px);
  }
`;

// --- COMPONENT IMPLEMENTATION ---
export const HeroParallax: React.FC = () => {
  const containerRef = useRef<HTMLElement>(null);
  const prefersReducedMotion = useReducedMotion();

  // Framer Motion Scroll Hooks
  const { scrollY } = useScroll();
  
  // Parallax mappings (Desktop only logic handled via CSS media queries disabling transform if needed, 
  // but Framer Motion values are calculated here safely)
  const bgY = useTransform(scrollY, [0, 800], [0, 150]);
  const contentY = useTransform(scrollY, [0, 800], [0, -100]);
  const contentOpacity = useTransform(scrollY, [0, 600], [1, 0]);

  return (
    <HeroContainer ref={containerRef}>
      <NebulaBackground 
        style={{ 
          y: prefersReducedMotion ? 0 : bgY 
        }} 
      />
      
      <ForegroundContent 
        style={{ 
          y: prefersReducedMotion ? 0 : contentY,
          opacity: contentOpacity
        }}
        initial={{ opacity: 0, y: 30 }}
        animate={{ opacity: 1, y: 0 }}
        transition={{ type: "spring", stiffness: 100, damping: 20, mass: 0.5, delay: 0.2 }}
      >
        {/* Typography components go here */}
        <PrimaryCTA>Start Your Evolution</PrimaryCTA>
      </ForegroundContent>
    </HeroContainer>
  );
};
```
