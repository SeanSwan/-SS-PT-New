## SESSION TITLE DISPLAY LOGIC

| Status | Client | Title Displayed |
|--------|--------|-----------------|
| available | null | "Available" |
| blocked | any | "Blocked" |
| booked | John Doe | "John Doe" |
| booked | null | "Booked" |
| confirmed | Jane Smith | "Jane Smith" |

**Implementation:**
```javascript
let title = 'Available';
if (session.status === 'blocked') title = 'Blocked';
else if (clientName) title = clientName;
else if (session.status !== 'available') title = 'Booked';
```

## ACCESSIBILITY

### Keyboard Navigation
- **Tab:** Navigate between controls
- **Arrow Keys:** Navigate calendar cells
- **Enter/Space:** Select date, open modal
- **Escape:** Close modal/dropdown

### ARIA Attributes
- `role="grid"` on calendar
- `role="gridcell"` on calendar cells
- `aria-selected` on selected date
- `aria-expanded` on dropdowns
- `aria-label` on all interactive elements
