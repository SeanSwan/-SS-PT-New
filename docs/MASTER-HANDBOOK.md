## Universal Master Schedule System

**Status:** Phase 0 design review in progress

**Documentation:**
- Business Logic: `docs/systems/UNIVERSAL-MASTER-SCHEDULE-HANDBOOK.md`
- Phase 0 Review: `docs/ai-workflow/reviews/universal-schedule-architecture.md`
- Registry: `docs/ai-workflow/PHASE-0-REGISTRY.md`

**Key Features:**
- One calendar with role-based views (admin, trainer, client).
- Recurring sessions via RFC 5545 recurrenceRule.
- Time blocking for trainer/admin availability.
- Notification opt-out per session action (notifyClient).
- Mobile-first Agenda view.
